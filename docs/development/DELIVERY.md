## 🚀 使用方法

### 快速开始（5分钟）

```bash
# 1. 进入项目目录
cd pdf-ocr-tool

# 2. 安装依赖
pip install -r requirements.txt

# 3. 配置AppCode
export ALIYUN_OCR_APPCODE="your_aliyun_appcode_here"

# 4. 运行测试
python test.py

# 5. 开始使用
python main.py
```
