## 📂 文件夹搜索逻辑

工具会在PDF所在目录下搜索包含**文件夹关键词**的文件夹。

### 搜索规则

- **子串匹配**：文件夹名称只需包含关键词即可
- **大小写敏感**：关键词区分大小写
- **优先匹配**：如果有多个文件夹包含同一关键词，使用第一个匹配的

### 示例

| 关键词 | 可匹配的文件夹名称 |
|--------|------------------|
| `测评授权书` | `8 测评授权书（word+pdf）`、`测评授权书`、`01测评授权书` |
| `风险告知书` | `9 风险告知书（word+pdf）`、`风险告知书`、`02风险告知书` |
| `测评调研表` | `3 测评调研表（Word+PDF）`、`测评调研表`、`03测评调研表` |
| `漏洞扫描报告` | `11漏洞扫描报告`、`漏洞扫描报告（word+pdf）` |
| `渗透测试报告` | `12渗透测试报告（word+pdf）`、`渗透测试报告` |
| `报告评审记录` | `16报告评审记录（Word+PDF）`、`报告评审记录` |

## ⚠️ 注意事项

### 1. 文件夹必须预先创建
工具不会自动创建文件夹，必须在运行前手动创建所需的文件夹。

### 2. 文件夹命名要求
- 文件夹名称必须包含对应的关键词
- 关键词区分大小写
- 可以在关键词前后添加编号、括号等内容

### 3. 多文档共享文件夹
以下文件夹会包含多种文档类型：

- **4 测评方案评审记录**：包含初审记录和复审记录
- **7 首末次会议记录**：包含启动会议和末次会议
- **10测评记录及问题汇总**：包含测评结果和问题列表
- **16报告评审记录**：包含报告初审、报告复审和项目验收评估表

### 4. 文件名冲突处理
如果目标文件夹中已存在同名文件，程序会自动添加后缀：
- `文件名.pdf`
- `文件名_1.pdf`
- `文件名_2.pdf`

## 🔧 自定义文件夹映射

如需修改文件夹映射，编辑 [doc_classifier.py](doc_classifier.py) 中的 `DOC_TYPE_TO_FOLDER` 字典：

```python
DOC_TYPE_TO_FOLDER = {
    "现场测评授权书": "测评授权书",  # 关键词
    "风险告知书": "风险告知书",      # 关键词
    # ...
}
```
