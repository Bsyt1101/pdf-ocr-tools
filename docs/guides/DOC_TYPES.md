## 🎯 命名规则说明

### 项目级文档
- **格式**: `文档类型.pdf`
- **特点**: 不带项目名，直接使用文档类型名称
- **示例**: `现场测评授权书.pdf`

### 系统级文档
- **格式**: `系统名-文档类型.pdf`
- **特点**: 自动将"xxx系统"替换为实际系统名
- **示例**: `门户网站系统-情况调查表确认签字.pdf`

## 🔍 文件编号格式

### 标准格式
```
JDB项目编号-类型码-系统编号
```

### 示例
```
JDB25300-CPSQS-01
JDB25300-DCB-02
JDB25245-FXGZS-01
```

### 组成说明
- **JDB25300**: 项目编号
- **CPSQS**: 文档类型简拼
- **01**: 系统编号（01=第1个系统，02=第2个系统）

## 💡 使用说明

### 1. 文件编号识别
工具会自动识别PDF中的文件编号（如：JDB25300-CPSQS-01），并根据类型码（CPSQS）确定文档类型。

### 2. 系统名替换
对于系统级文档，工具会自动将"xxx系统"替换为实际的系统名称。

例如：
- 文档类型: `xxx系统-情况调查表确认签字`
- 系统名称: `门户网站系统`
- 生成文件名: `门户网站系统-情况调查表确认签字.pdf`

### 3. 项目级文档
项目级文档不带项目名，直接使用文档类型名称作为文件名。

例如：
- 文档类型: `现场测评授权书`
- 生成文件名: `现场测评授权书.pdf`

### 4. 无页码后缀
所有文件名都不添加页码后缀，即使同类型有多页文档。
